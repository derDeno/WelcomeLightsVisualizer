# WelcomeLightsVisualizer
Tool to create welcome lights animations for BMW cars with the FLM2 module

Live site: https://derdeno.github.io/WelcomeLightsVisualizer/


## How to use
Visit the live site link above and you are ready to go.


## Adding new templates
If you have a custom template you want to share with others or want to be part of the tool, follow these steps:

1. go to the issues page
2. click new issue
3. select the "New Template Submission" option
4. fill all required fields
5. submit
